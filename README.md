VERSION/DATE: 1.0 - May 2019
AUTHOR: Vachia Thoj
————

—— If you have Greenfoot installed —— 
Clone/download the project, then go into the "TicTacToe" folder.
There should be a file called "project.greenfoot" or "project". Click on this file to open the game in greenfoot. Then compile and run
the program.

—— If you have JDK 8 or newer installed (Running game with jar file) ——
There is an executable jar file called "TicTacToe.jar". Once the project has been downloaded, by clicking on this jar file, it
will run the game.
